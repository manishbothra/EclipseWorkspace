HS Code Finder

The Harmonized Commodity Description and Coding System is a standard system of names and numbers adopted internationally to classify products for the purpose of trade. It was developed by World Customs Organization (an independent intergovernmental organization based in Brussels, Belgium, with over 200 member countries.) in 1998 and is also being currently maintained by this organisation.
